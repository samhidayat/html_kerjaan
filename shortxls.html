Tombol Shortcut Keterangan Fungsi
Ctrl + 1 Menampilkan kotak dialog Format Sel
Ctrl + Shift + ~ Mengubah Sel Excel menjadi Format General
Ctrl + Shift + $ Mengubah Sel Excel menjadi Format Currency
Ctrl + Shift + % Mengubah Sel Excel menjadi Format Persentase
Ctrl + Shift + ^ Mengubah Sel Excel menjadi Format Scientific
Ctrl + Shift + # Mengubah Sel Excel menjadi Format Tanggal (date)
Ctrl + Shift + @ Mengubah Sel Excel menjadi Format Waktu (time)
Ctrl + Shift + ! Mengubah Sel Excel menjadi Format Number (angka)
